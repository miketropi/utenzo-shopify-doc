## General Questions

### What is the Utenzo Shopify theme?

Utenzo is a premium Shopify theme designed for modern e-commerce stores. It features a clean, responsive design with advanced customization options, perfect for fashion, electronics, home goods, and general merchandise stores.

### Is Utenzo mobile-responsive?

Yes, Utenzo is fully responsive and optimized for all devices including mobile phones, tablets, and desktops. The theme automatically adapts to different screen sizes to ensure your store looks great on any device.

### Which Shopify plan do I need?

Utenzo works with all Shopify plans including Basic Shopify, Shopify, and Advanced Shopify. However, some advanced features may require specific Shopify plan features.

## Installation & Setup

### How do I install the Utenzo theme?

1. Download the theme ZIP file
2. Log into your Shopify admin
3. Go to **Online Store > Themes**
4. Click **Upload theme**
5. Select the ZIP file and upload
6. Once uploaded, click **Actions > Publish**

For detailed instructions, see our [Installation Guide](/docs/getting-started/installation).

### Do I need coding knowledge to use Utenzo?

No coding knowledge is required! Utenzo comes with an intuitive theme customizer that allows you to modify colors, fonts, layouts, and other settings through Shopify's visual editor.

### Can I customize the theme colors and fonts?

Yes, Utenzo includes extensive customization options:
- **Colors**: Customize primary, secondary, accent colors, and more
- **Typography**: Choose from Google Fonts and adjust font sizes
- **Layout**: Modify spacing, container widths, and section layouts
- **Buttons**: Customize button styles, colors, and hover effects

### Will I lose my customizations if I update the theme?

Theme updates may overwrite your customizations. We recommend:
- Using the theme customizer for styling changes
- Keeping a backup before updating
- Documenting any custom code modifications

## Troubleshooting

### Why isn't my theme displaying correctly?

Common solutions:
1. **Clear browser cache** and refresh the page
2. **Check theme activation** - ensure Utenzo is your published theme
3. **Verify file upload** - all theme files uploaded correctly
4. **Review customizations** - recent changes may have caused issues

### My images look blurry or pixelated. Why?

To ensure crisp images:
- Upload high-resolution images (at least 1200px wide)
- Use recommended image formats (JPG, PNG, WebP)
- Enable image optimization in theme settings
- Check your image compression settings

### The theme is loading slowly. How can I fix this?

Speed optimization tips:
1. **Optimize images** - compress and resize images
2. **Remove unused apps** - deactivate apps you don't need
3. **Minimize customizations** - excessive custom code can slow things down
4. **Use a CDN** - consider Shopify's CDN or third-party options

### Can I revert to my old theme?

Yes, your previous themes remain available:
1. Go to **Online Store > Themes**
2. Find your old theme in the theme library
3. Click **Actions > Publish** to revert

  Always backup your current theme before making major changes or switching themes.
